#########################  ORDER DELETE Prestashop MODULE  #########################

= Installation =

1. Copy the content of the downloaded zip file in the modules folder, or upload the entire zip from your back office
2. Open the Administration tab, find Order Delete and install it
3. Back up your database before deleting order, then insert the id where you want to start deleting from, anche click Delete!
4. All of the unwanted orders and related data should have been erased!
